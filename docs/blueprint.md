# **App Name**: CampusConnect

## Core Features:

- User Authentication: Secure student and admin login using Firebase Authentication with role-based access control.
- College Browsing: Browse colleges with filtering by region, fees, ranking, and eligibility.
- Aptitude Test: Take a timed aptitude test with verbal, quantitative, and general knowledge sections. The UI has Multiple-Choice Questions (MCQ).
- Automated College Recommendation Tool: Based on CGPA, exam scores, aptitude test, preferences, and AI. It will analyze your data and find a university or college you may want to apply for.
- Registration: Student-college registration system where students apply and upload documents.
- Admin Dashboard: Admin dashboard for managing colleges, student applications, and aptitude tests.
- College Profile Management: Add, edit, and store the university and college brochures, fees and upload them in the Firestore storage.

## Style Guidelines:

- Primary color: #4CAF50 (Green) to symbolize growth and learning.
- Background color: #F0F5F1 (Light Green), a desaturated shade of green providing a calm backdrop.
- Accent color: #3F51B5 (Indigo), to draw attention to calls to action, provide balance to the green, and evoke credibility.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, 'Inter' (sans-serif) for body text. For computer code snippets: 'Source Code Pro'.
- Use consistent, professional icons from a library like Material UI to represent actions and data.
- Fully responsive design with a clean, intuitive layout, prioritizing easy navigation and data presentation.
- Subtle transitions and animations to enhance user experience without overwhelming.
